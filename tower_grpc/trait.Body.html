Provided Methods

fn is_end_stream(&self) -> bool

Returns true when the end of the stream has been reached.

An end of stream means that both poll_data and poll_metadata will return None.

A return value of false does not guarantee tht a value will be returned from poll_data or `poll_trailers. This is merely a hint.
